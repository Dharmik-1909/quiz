# Quiz App

## Features

- Multiple Choice Questions (MCQ)
- Previous and Next buttons
- Submit Answer
- 1 Minute Timer
- Correct and Wrong Answer Count
- Result Screen
- Percentage Calculation

---

## Technologies Used

- HTML5
- CSS3
- JavaScript

---

## Quiz Features

- Every question has 4 options.
- Only one option can be selected.
- Timer starts automatically.
- Correct answer increases the score.
- Wrong answer increases the wrong answer count.
- After all questions are submitted, the result is displayed.

---

## Result Screen

The result page displays:

- Total Correct Answers
- Total Wrong Answers
- Percentage

Example:

```
Quiz Result

Correct Answers : 8

Wrong Answers : 2

Percentage : 80%
```
